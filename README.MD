### 缺点与不足

1. 顶级函数缺乏命名空间，这让我们在读相关代码时很难快速定位用的是哪一个函数。
2. 空安全并非绝对安全，在与Java互操作过程中可能仍会发生异常。
3. 与java的互调用语法不友好，容易出错。
4. 非受检异常，这意味着编译器不会强迫你捕获其中的任何一个，但这样做也许会导致运行时抛出异常。
